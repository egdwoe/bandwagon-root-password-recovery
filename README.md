## 一、搞清楚"root 密码"是什么

刚接触 VPS 的朋友可能有点懵：root 密码、KiwiVM 密码、搬瓦工账号密码，这三个东西到底有什么区别？

简单说：

- **搬瓦工账号密码**：登录 bwh81.net 官网用的，管理订单、续费、开工单都在这里。
- **KiwiVM 面板密码**：可选的，用于单独登录 KiwiVM 控制台（一般不需要单独设置，直接从官网跳转进去就行）。
- **root 密码（SSH 密码）**：用 SSH 工具连接你的 VPS 服务器时需要输入的密码，也就是本文的主角。

你找不到、忘记了、或者想更新的，就是这个 root 密码。

## 二、准备工作

在开始重置之前，需要确认两件事：

**① 你能正常登录搬瓦工官网账号**

如果连账号都进不去，那先找回账号密码。官网的"Forgot Password"功能，输入注册邮箱，收邮件点链接重置即可。这步和 root 密码无关，跳过不说。

**② 你知道自己 VPS 的服务页面在哪里**

登录搬瓦工官网后，点击顶部导航 **Services → My Services**，就能看到你购买的 VPS 列表。

## 三、搬瓦工找回 root 密码的完整步骤

### 第一步：进入 KiwiVM 控制面板

登录 👉 [搬瓦工官网](https://bwh81.net/aff.php?aff=77528)，在 **My Services** 页面找到你的 VPS，点击右侧的 **KiwiVM Control Panel** 按钮，跳转进入该 VPS 专属的 KiwiVM 控制台。

> 💡 如果你有多台 VPS，注意确认进入的是正确那一台的 KiwiVM，别搞混了。

---

### 第二步：确认 VPS 处于运行状态

进入 KiwiVM 后，首先看一下 VPS 的当前状态。

- 如果显示 **Running**，直接进行下一步。
- 如果显示 **Stopped**（已停机），点击 **Start** 按钮开机，等待 15～30 秒，等状态变为 Running 再继续。

**注意**：这一步很多人容易忽略，直接去重置密码结果报错 `Failed to reset root password (739102)`，原因就是 VPS 没有开机。**重置 root 密码需要 VPS 处于运行状态**，这是目前搬瓦工 KiwiVM 的机制。

---

### 第三步：点击 Root Password Modification

在 KiwiVM 面板左侧菜单往下翻，找到 **Root password modification**（部分界面显示为 Root password reset），点击进入。

进入后，你会看到一个蓝色按钮：

> **Generate and set new root password**

点击这个按钮，系统会自动生成一个新的随机密码并立即生效。

**⚠️ 划重点：新密码只显示一次！**

重置成功后，页面会展示你的新 root 密码。这个密码只在这个页面出现一次，关掉页面就消失了（系统也会发一封邮件，但邮件可能延迟或进垃圾箱）。

所以，看到新密码的第一件事：**复制，粘贴到安全的地方保存好。**

## 四、拿到新密码之后怎么连接 VPS？

拿到新的 root 密码，接下来就可以用 SSH 工具登录了。连接需要三个信息：

| 信息 | 在哪里找 |
|------|----------|
| IP 地址 | KiwiVM 主页面顶部显示 |
| SSH 端口 | KiwiVM 主页面（不是默认的 22，是随机生成的，通常是五位数） |
| root 密码 | 刚才重置拿到的 |

用 Windows 可以用 **Xshell**、**PuTTY** 或者 **MobaXterm**；Mac/Linux 直接终端 `ssh root@你的IP -p 端口号` 就行；手机端推荐 **Termius**（iOS/Android 都有）。

## 五、想把密码改成自己好记的？

KiwiVM 重置的密码是随机字符串，长得像 `aK7#mX9p2Lz`，很难记。如果你想换一个自己好记的密码，有两种方法：

### 方法一：登录 SSH 后用 `passwd` 命令修改

用新密码登录 VPS 之后，直接在命令行输入：

bash
passwd


然后按提示输入新密码（输入时不会显示字符，这是正常的），回车确认，搞定。

### 方法二：在 KiwiVM 的 Interactive Root Shell 里改

如果 SSH 还没连上，可以直接在 KiwiVM 里用 **Root shell - Basic** 功能，输入以下命令：

bash
echo "root:你想要的新密码" | chpasswd


注意：`root:` 后面跟英文冒号，没有空格，把 `你想要的新密码` 替换成实际想设的密码。

## 六、常见问题 Q&A

**Q：点击重置密码后报错 "VPS is currently stopped"，怎么办？**

A：先回主页面把 VPS 开机（点 Start），等 Running 之后再去重置。

**Q：密码重置成功了但 SSH 还是登不上，怎么回事？**

A：几个可能原因——①端口输错了，搬瓦工的 SSH 端口不是 22，注意确认；②输入密码时有多余空格；③VPS 的防火墙把端口封了，可以用 KiwiVM 的 Interactive Root Shell 先检查一下。

**Q：重置密码的邮件没收到？**

A：邮件有延迟是正常的，而且很多邮箱（QQ 邮箱尤其明显）会把搬瓦工的邮件推进垃圾箱。建议换用 Gmail 注册搬瓦工账号，邮件接收更稳定。

**Q：找不到 Root password modification 这个菜单？**

A：KiwiVM 在 2024 年改版过一次界面，有些功能菜单的位置挪了，往下翻仔细找一下，或者搜索一下，功能依然存在。
